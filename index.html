💡 使用技巧

智能批量生成圖片：
• 支援「標籤: 網址」格式 (例如：** 專案: https://example.com)
• 支援「標籤,網址」格式 (例如：項目01,https://example.com)
• 自動偵測並生成對應的QR Code

PDF生成問題：
• 確保圖片格式正確 (JPG/PNG)
• 避免使用過大的圖片檔案
• 嘗試重新上傳問題圖片

最佳效果建議：
• 使用高對比度的QR Code
• 確保QR Code大小適中
• 在充足光線下進行掃描
